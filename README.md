# office-light-trigger
Code tested on Intel Galileo, with servo, and photoresistor

```///////////////////////////////////////////////////////
// By: Je'aime Powell 
// Rev 1 - 9/10/19
//
// Office Light Trigger
//
// Activates servo to wave arm to turn back on the motion
// controlled light
//
// Pin Connections:
// D13 - Intel Galileo LED Pin
// D9 - Servo
// A1 - Photoresistor
///////////////////////////////////////////////////////```
